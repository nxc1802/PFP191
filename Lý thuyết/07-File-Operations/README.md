# Thao Tác File (File Operations) trong Python

## Nội dung

### 1. Mở và đóng file
- `open()`, `close()`
- Mode: `r`, `w`, `a`, `r+`
- Dùng `with` (khuyên dùng)

### 2. Đọc file
- `read()`, `readline()`, `readlines()`
- Đọc từng dòng

### 3. Ghi file
- `write()`, `writelines()`
- Mode `w` (ghi đè) vs `a` (nối thêm)

### 4. Xử lý lỗi file
- `FileNotFoundError`
- Kiểm tra file tồn tại
