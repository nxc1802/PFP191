# Câu Lệnh Điều Kiện trong Python

## Nội dung

### 1. if / elif / else
- Cú pháp cơ bản
- if lồng nhau
- Thứ tự elif quan trọng

### 2. Toán tử 3 ngôi (Ternary Operator)
```python
kết_quả = giá_trị_true if điều_kiện else giá_trị_false
```

### 3. Xử lý lỗi: try / except
- `try-except` cơ bản
- Bắt nhiều loại lỗi
- `try-except-else-finally`
- Các loại lỗi thường gặp: ValueError, ZeroDivisionError, IndexError, TypeError, NameError
