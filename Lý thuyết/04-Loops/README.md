# Vòng Lặp (Loops) trong Python

## Nội dung

### 1. Vòng lặp for
- `range(n)`, `range(start, stop)`, `range(start, stop, step)`
- Lặp qua list, string
- `enumerate()` lấy cả index

### 2. Vòng lặp while
- Cú pháp cơ bản
- Tránh vòng lặp vô hạn
- Khi nào dùng for vs while

### 3. break và continue
- `break` - thoát vòng lặp
- `continue` - bỏ qua lần lặp hiện tại
- `for-else` / `while-else`

### 4. Vòng lặp lồng nhau
- Bảng cửu chương
- In hình tam giác sao

### 5. Pattern đề thi
- Tính tổng, đếm, tìm min/max, xây dựng list mới
