### 2. Prints a list of Armstrong numbers sorted in the file.
An Armstrong number is a number that is equal to the sum of its own digits raised to the power of the number of digits eg. $153 = 1^3 + 5^3 + 3^3 = 1 + 125 + 27 = 153$.

**The output should be:**
```
Your selection (1 -> 2): 2
data.txt
OUTPUT
[1, 153, 370, 371, 407]
FINISH
```
